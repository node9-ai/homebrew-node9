# homebrew-node9

[![License: Apache-2.0](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)

Homebrew tap for [Node9](https://github.com/node9-ai/node9-proxy) — the execution security layer for AI agents (Claude Code, Gemini CLI, Cursor, MCP Servers).

## Install

```bash
brew tap node9-ai/node9
brew install node9
```

## Upgrade

```bash
brew upgrade node9
```

## Verify

```bash
node9 doctor
```
